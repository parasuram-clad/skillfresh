Weeks 1–2

User Research

Understand users through surveys, interviews, and personas. Learn empathy-driven design principles.

Weeks 3–4

Wireframing & Layouts

Create low-fidelity sketches and digital wireframes. Plan user flows and page structures for apps and websites.

Weeks 5–7

Prototyping

Build interactive prototypes using Figma/Adobe XD to simulate real product experiences and validate flows early.

Weeks 8–9

Usability Testing

Test your designs with real users, gather feedback, and iterate for improved usability and accessibility.

Weeks 10–12

Design Systems & AI Tools

Learn how to build scalable design systems and style guides. Explore AI-powered tools (like Figma AI, Galileo AI, Uizard) to speed up ideation, generate layouts, and enhance creativity. Wrap up with a portfolio-ready project.
